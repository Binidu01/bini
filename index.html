Type-Safe API Routes

Create powerful endpoints with zero boilerplate

Basic TypeScript API

Simple endpoints with automatic JSON serialization and type safety.

// src/app/api/users.ts import type { Request, Response } from 'fastify' export default function handler( req: Request, res: Response ) { return { users: [] } }

Dynamic Routes

URL parameters with full type safety. Supports [id] and [...slug].

// src/app/api/products/[id].ts export default function handler(req, res) { const { id } = req.query return { product: { id, name: `Product ${id}` } } }

Request Methods

Full REST support with GET, POST, PUT, DELETE, PATCH.

export default function handler(req, res) { switch (req.method) { case 'GET': return { data: [] } case 'POST': res.status(201) return { created: true } default: res.status(405) } }

How do I create API routes?

Create files in src/app/api/:

  • src/app/api/hello.ts → /api/hello
  • src/app/api/users.ts → /api/users
  • src/app/api/posts/[id].ts → /api/posts/:id

Each handler exports a default function that receives (req, res) and returns JSON.
