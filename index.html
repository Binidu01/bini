How do I create custom 404 pages?

Create src/app/not-found.tsx or src/app/not-found.jsx:

// src/app/not-found.tsx export default function NotFound() { return ( <div style={{ minHeight: '100vh', display: 'flex', alignItems: 'center', justifyContent: 'center' }}> <h1>404 - Page Not Found</h1> </div> ) }

Bini.js automatically detects this file and uses it for all 404 responses.
